Glosarium Coding & AI

Kecerdasan Buatan (AI)

Sistem komputer yang mampu meniru fungsi kognitif manusia seperti belajar, bernalar, dan memecahkan masalah.

Pembelajaran Mesin (Machine Learning/ML)

Sub-bidang AI yang fokus pada pengembangan algoritma yang memungkinkan komputer "belajar" dari data tanpa diprogram secara eksplisit.

Jaringan Saraf Tiruan (Neural Network)

Model komputasi yang terinspirasi oleh struktur otak manusia, terdiri dari lapisan-lapisan node (neuron) yang memproses data.

Pembelajaran Mendalam (Deep Learning)

Jenis ML yang menggunakan jaringan saraf tiruan dengan banyak lapisan (deep neural networks) untuk menganalisis data yang kompleks (seperti gambar, teks, suara).

Model Bahasa Besar (Large Language Model/LLM)

Model Deep Learning yang dilatih pada data teks dalam jumlah sangat besar, mampu memahami, menghasilkan, dan merespons bahasa manusia (contoh: GPT, Gemini).

Pelatihan (Training)

Proses memasukkan data ke dalam model AI/ML agar model tersebut dapat belajar dan menyesuaikan bobotnya untuk membuat prediksi.

Hyperparameter Tuning

Proses memilih nilai optimal untuk variabel yang mengontrol proses pelatihan itu sendiri (bukan parameter model yang dipelajari dari data), seperti learning rate atau jumlah lapisan pada neural network.

Bias (dalam AI)

Kecenderungan model AI untuk menghasilkan hasil yang tidak adil atau diskriminatif karena adanya prasangka atau representasi yang tidak seimbang dalam data pelatihan yang digunakan. Ini adalah isu etika yang sangat penting.

Data Set

Kumpulan data terstruktur yang digunakan untuk melatih, menguji, dan memvalidasi model Pembelajaran Mesin. Data Set adalah bahan bakar utama bagi AI.

Reinforcement Learning (RL)

Jenis Pembelajaran Mesin di mana sebuah agen belajar untuk mengambil serangkaian keputusan untuk memaksimalkan "hadiah" (reward) dalam lingkungan tertentu. Ini sering digunakan untuk melatih AI dalam bermain game atau mengontrol robot.

Overfitting

Sebuah fenomena dalam Pembelajaran Mesin di mana model belajar detail dan noise dari data pelatihan hingga tingkat yang tidak relevan, sehingga performanya sangat baik pada data pelatihan, tetapi sangat buruk pada data baru yang belum pernah dilihat. Ibarat terlalu menghafal soal ujian, tapi gagal saat menghadapi soal baru.

Algoritma

Serangkaian instruksi atau aturan terperinci dan terbatas yang digunakan untuk memecahkan masalah atau menyelesaikan suatu tugas.

Sintaks (Syntax)

Aturan tata bahasa yang menentukan bagaimana pernyataan dalam bahasa pemrograman harus ditulis agar dapat dimengerti oleh kompiler atau interpreter.

Variabel (Variable)

Lokasi penyimpanan di memori yang diberi nama. Digunakan untuk menyimpan nilai data yang dapat diubah selama eksekusi program.

Indensasi (Indentation)

Penggunaan spasi kosong atau tab di awal baris kode untuk menunjukkan struktur hierarki dan cakupan dari blok kode. Indensasi wajib di Python, namun hanya sebagai gaya di bahasa lain seperti C++ atau Java.

Fungsi (Function)

Blok kode yang dirancang untuk melakukan tugas tertentu. Digunakan untuk mengorganisir kode dan membuatnya dapat digunakan kembali (reusable).

Debug (Debugging)

Proses menemukan dan memperbaiki kesalahan (bug) atau error dalam kode program.

API (Application Programming Interface)

Sekumpulan definisi dan protokol yang memungkinkan dua aplikasi perangkat lunak berkomunikasi satu sama lain.

Kompiler (Compiler)

Program yang menerjemahkan seluruh kode sumber dari bahasa tingkat tinggi ke dalam bahasa mesin sebelum program dijalankan.

Interpreter

Program yang menerjemahkan dan menjalankan kode program baris demi baris saat program sedang berjalan.

Kode Sumber (Source Code)

Sekumpulan instruksi pemrograman yang ditulis oleh manusia dalam bahasa pemrograman.

Library

Kumpulan kode yang sudah ditulis sebelumnya dan dapat dipanggil oleh programmer untuk melakukan tugas tertentu tanpa harus menulis kode dari awal. Contoh: NumPy untuk komputasi numerik atau Matplotlib untuk visualisasi data.

Framework

Sekumpulan abstraksi perangkat lunak yang menyediakan fungsi generik (seperti struktur, library, dan aturan) untuk mempercepat pengembangan aplikasi. Contoh: Django (Python) atau React (JavaScript).

Git & GitHub (Version Control)

Sistem yang melacak dan mengelola perubahan pada kode sumber dari waktu ke waktu. Git adalah tool lokal, sedangkan GitHub (atau GitLab/Bitbucket) adalah platform online untuk kolaborasi dan penyimpanan kode.

Object-Oriented Programming (OOP)

‚Äã Paradigma pemrograman yang mengatur desain perangkat lunak di sekitar objek, daripada fungsi dan logika. OOP fokus pada empat konsep utama: Encapsulation, Inheritance, Abstraction, dan Polymorphism.

Unit Test

Proses pengujian bagian terkecil (unit) dari sebuah aplikasi (seperti fungsi atau metode tunggal) untuk memastikan bahwa setiap komponen kode berjalan seperti yang diharapkan. Ini kunci untuk debug yang efektif.

Dasar Artificial Intelligence

                        

                            
Pengertian AI

                            
Artificial Intelligence (AI) adalah cabang ilmu komputer yang berfokus pada pembuatan mesin yang dapat melakukan tugas-tugas yang biasanya memerlukan kecerdasan manusia.

                        

                        

                            
Jenis-jenis AI

                            
    
                                
  • AI Lemah (Weak AI): Dirancang untuk tugas spesifik (contoh: asisten virtual)
    • 
                                
  • AI Kuat (Strong AI): Memiliki kemampuan kognitif setara manusia
    • 
                                
  • Superintelligence: Melebihi kecerdasan manusia dalam semua bidang
    •

  • Spesifik dan Jelas: "Buatkan kode Python untuk menghitung luas persegi panjang"

    • 
                                
  • Gunakan Contoh: "Beri 3 contoh implementasi loop dalam JavaScript"
    • 
                                
  • Tentukan Format: "Jelaskan dalam bentuk poin-poin singkat"

Variabel dan Tipe Data

                            
nama = "Budi"          # string

tinggi = 170.5        # float

Operator

                            
# Aritmatika
hasil = 10 + 5        # Penjumlahan
hasil = 10 * 2        # Perkalian

# Perbandingan

is_lebih_besar = (10 > 5)  # True

                        

                        

                            
Struktur Kontrol

    print("Dewasa")

# Loop
for i in range(5):
    print(i)
